## Getting Started

### Prerequisites

Ensure you have the following installed:

- NPM
- Yarn

### Installation

1. Install dependencies:
   ```bash
   yarn install
   ```

### Running the Application

To start the application, run:

```bash
yarn run dev
```

You can access the application via your web browser at http://localhost:5173/.
